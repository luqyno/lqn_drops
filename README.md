# lqn_drops

**lqn_drops** is a simple standalone FiveM script that allows players to pick up and re-equip dropped accessories like hats or glasses.

When an accessory falls off your character (for example after a melee attack), it will appear on the ground and can be picked up again using **target interaction** and instantly re-equipped.

The script is **framework-independent** and works on **all FiveM frameworks**.

---

## ✨ Features
- Dropped hats & glasses can be picked up from the ground  
- Uses target interaction (clean & immersive)  
- Automatically re-equips the accessory  
- Standalone (no framework required)  
- Lightweight and easy to install  

---

## 📦 Dependencies
- [ox_lib](https://github.com/overextended/ox_lib)
- [ox_target](https://github.com/overextended/ox_target)

---

## ⚙️ Installation
1. Make sure you have **ox_lib** and **ox_target** installed and running.
2. Add `lqn_drops` to your server resources folder.
3. Restart your server.

✅ Done!
